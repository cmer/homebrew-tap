# cmer/homebrew-tap

Homebrew tap for Carl Mercier's applications.

## Usage

```sh
brew tap cmer/tap
brew install --cask gateopener
```

Or in a single command without tapping first:

```sh
brew install --cask cmer/tap/gateopener
```

## Available casks

| Cask | Description |
| ---- | ----------- |
| [gateopener](Casks/gateopener.rb) | Removes Gatekeeper quarantine attributes from apps |
